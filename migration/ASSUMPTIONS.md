# Assumptions Log

Record assumptions the agent makes to keep the loop moving when product intent is unclear.

This is **not** a stop condition. The agent must proceed, but must document assumptions here (and link to the
relevant node/iteration in the tree/logs).

## Template

### YYYY-MM-DD — Context / node id

- **Assumption:** what we’re assuming is true
- **Reason:** why we made this assumption
- **Risk:** what could be wrong / impact if false
- **How to validate:** what human answer or check would confirm
